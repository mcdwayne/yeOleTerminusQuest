# WELCOME TO YE OLE TERMINUS QUEST

## A text based adventure game for learning Terminus

## Learning Terminus has never been more like Zork.  

## Get ready to do what a Wizard says, meet robots and rescue the Terminus art!

## To to play:

1. Download `yeOleTerminusQuest.php`
2. Open your terminal
3. `cd` to wherever you put this file
4. run `php yeOleTerminusQuest.php`
